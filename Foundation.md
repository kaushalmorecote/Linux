## Lesson 1: What is a Computer?

A **computer** is an **electronic machine** that does these four things:

- Takes **input** (you give it something)
- **Processes** it (does work/calculation)
- Gives **output** (shows result)
- Can **store** data for later

**Example**:  
Calculator → 5 + 3 = 8  
Your phone, laptop, ATM, smart TV — all are computers.

## Lesson 4: How a Computer Works (Simple Flow)

1. Press Power Button → Electricity starts.
2. Small program on motherboard (**BIOS**) wakes up.
3. Loads **Operating System** (Windows/Linux) from Hard Disk into RAM.
4. OS starts and shows Desktop.
5. When you open Chrome:
   - OS copies Chrome from Hard Disk → RAM
   - CPU starts running it
   - You see it on screen

**Golden Rule**:  
**Hard Disk → RAM → CPU**  
(Data moves from slow storage → fast memory → brain)

## Lesson 6: Bit, Byte and Storage Units

Computer only understands **0 and 1** (called **Binary**).

- 1 **Bit** = 0 or 1
- 8 Bits = 1 **Byte**
- 1024 Bytes = 1 **KB**
- 1024 KB = 1 **MB**
- 1024 MB = 1 **GB**
- 1024 GB = 1 **TB**

**Examples**:
- One photo ≈ 5 MB
- One movie ≈ 1–2 GB

## Lesson 7: What is Software?

Software = Set of instructions for the computer.

### Two Types:
1. **System Software** → Operating System (Windows, Linux, Android)
2. **Application Software** → Apps (Chrome, WhatsApp, MS Word, Games)
