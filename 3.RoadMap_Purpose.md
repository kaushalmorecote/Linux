## 3. Complete Subject-Wise Roadmap (Pareto 80/20 + Zero Compromise on Foundation)

### Phase 0: Mindset (1–2 days)
- Understand daily job problems and how the foundation fixes them fast.
- Rule: Never move forward until you understand 100%.

### 1. Linux + Shell/CLI Mastery (Weeks 1–3)
**80/20 Focus:** Processes, logs, permissions, networking commands (`ss`, `curl`, `dig`), systemd, troubleshooting.  
**Practice:** Break & fix a Linux VM.

### 2. Networking & Security Basics (Weeks 3–4)
**80/20 Focus:** DNS, HTTP/HTTPS/TLS, firewalls, SSH, VPC, OSI layers 3–7.  
**Practice:** Nginx + SSL + firewall setup.

### 3. Git + Version Control (Weeks 4–5)
**80/20 Focus:** Branching, merge/rebase, conflicts, GitOps mindset.  
**Practice:** Intentional merge conflicts.

### 4. Scripting (Bash + Python basics) (Weeks 5–6)
**80/20 Focus:** Loops, functions, cron, JSON/API handling.  
**Practice:** 5–10 scripts daily.

### 5. Docker & Containers (Weeks 6–8)
**80/20 Focus:** Dockerfile best practices, multi-stage, volumes, docker-compose, debugging.  
**Practice:** Build, break, fix images.

### 6. CI/CD Pipelines – GitHub Actions (Weeks 8–10)
**80/20 Focus:** Pipeline-as-code, triggers, caching, secrets, full build-test-deploy.

### 7. Kubernetes (Weeks 10–13)
**80/20 Focus:** Pods, Deployments, Services, ConfigMaps/Secrets, Ingress, debugging, basic Helm.  
**Practice:** Minikube/Killercoda.

### 8. Infrastructure as Code – Terraform (Weeks 13–15)
**80/20 Focus:** Modules, state, variables, real cloud resources.

### 9. Cloud – AWS (One Cloud Only) (Weeks 15–17)
**80/20 Focus:** EC2, VPC, S3, RDS, IAM, Load Balancers, Route53, Auto Scaling.

### 10. Observability + DevSecOps Basics (Weeks 17–20)
**80/20 Focus:** Prometheus + Grafana, logging, secrets management, Trivy scanning, RBAC.

**Final Milestone Project (End of Week 20):**  
Full-stack app using Terraform (AWS) → Docker → GitHub Actions CI/CD → Kubernetes → Prometheus/Grafana monitoring.
